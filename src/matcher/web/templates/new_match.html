1. 選擇範本

範本決定要配對的角色種類（如老師、學生）和條件規則。沒看到想要的？建立新範本 →

2. 上傳名單

檔案第一列需是欄位名稱；可用中文也可用英文（依範本設定的別名）。

不會做名單？下載範例： 教師-班級範例（CSV） · 研習分組範例（Excel）

3. 設定亂數種子

亂數種子是什麼？抽籤需要亂數，這個整數決定了「這次的隨機順序」。 重點：同一份名單 + 同一個種子 → 永遠抽出一樣的結果，方便事後重現給家長 / 教評委員看。 建議公開抽籤現場以擲骰、彩券號碼等公開方式決定種子數字，讓所有人都信服。

4. 選抽籤方式

  • 純抽籤：所有人不填志願、完全公平隨機（適合：分組順序無偏好的場景）
  • 輪流挑：按隨機順序，每個人輪流挑自己最高志願（適合：研習分組、社團選填）
  • 依志願先後填滿：先把每組的第一志願填滿，剩下的人再填第二志願⋯（適合：志願清楚分高低的場景）
